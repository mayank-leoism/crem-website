# My Static Site

This is a simple static site project that demonstrates the use of HTML and CSS to create a basic webpage.

## Project Structure

```
my-static-site
├── index.html
├── styles
│   └── main.css
└── README.md
```

- **index.html**: The main HTML document for the static site.
- **styles/main.css**: The CSS file that contains styles for the site.

## Getting Started

To set up and run this static site, follow these steps:

1. **Clone the repository** (if applicable):
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd my-static-site
   ```

3. **Open the `index.html` file in your web browser**:
   You can simply double-click the `index.html` file or open it using your preferred web browser.

## Customization

Feel free to modify the `index.html` and `styles/main.css` files to customize the content and appearance of the site according to your needs.
